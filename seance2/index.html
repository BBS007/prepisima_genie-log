Structurer un document

Titre et auteurs

Définition du titre : 
            \title{Mon super titre}
Définition des auteurs : 
            \author{Moi bien sûr}
Afficher le tout : 
            \maketitle

Structurer un document

Définir des sections

Définition d'une section : 
              \section{Mon super titre de section}
Définition d'une sous section : 
              \subsection{Mon super titre de sous-section}
Générer la table des matières correspondante : 
              \tableofcontents

Notations mathématiques de base

  • $ : entrer/sortir du mode maths
  • _ : mettre en indice
  • ^ : mettre en exposant

Attention à ces caractères hors du mode maths !

Exemple :

            $x_{2}$
            $y^{4}$
            $f_{0}^{x}$

Gestion des images

  • \includegraphics[ < options > ]{ filepath }
  • environnement figure
  • Légende : \caption{Légende}
  • Numérotation : \label{identifiant}
  • Citer dans le texte : \ref{identifiant}
  •             \begin{figure}[h]
            \includegraphics[keepaspectratio,scale=0.4]{image.png}
            \caption{Légende apportant des informations}
            \label{mafigure}
            \end{figure}
            ...
            Comme dans la figure \ref{mafigure}
